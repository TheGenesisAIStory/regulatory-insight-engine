# Release Scope Note

Fiorell.IA should launch with a narrow and credible beta claim.

Recommended wording:

```text
Fiorell.IA is a local Italian-first assistant for inspecting selected banking regulatory sources. The current beta is strongest on default, internal controls, own funds and prudential supervision. Coverage of broader IFRS 9, Pillar 3, EBA/Basel consolidated frameworks and bank-specific disclosures depends on the local corpus and should be validated before use.
```

Do not say:

```text
Fiorell.IA covers all banking regulation.
```

Do not imply that Fiorell.IA replaces professional regulatory, legal, accounting or supervisory review.

Do not imply that Fiorell.IA owns runtime thresholds, gate modes, retrieval settings or serving behavior. Those remain part of the shared runtime.
