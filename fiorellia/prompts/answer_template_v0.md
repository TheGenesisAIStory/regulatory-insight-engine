# Fiorell.IA Answer Template v0

Compact Italian template for grounded answers. This is a product/evaluation artifact and is not wired into the shared runtime by itself.

```text
Risposta:
<risposta breve, tecnica e limitata ai fatti supportati dal contesto recuperato>

Fonti:
- <documento locale> — <pagina/sezione/articolo se disponibile>
- <documento locale> — <pagina/sezione/articolo se disponibile>

Nota:
Risposta limitata ai documenti indicizzati nel corpus locale. Se servono copertura piu' ampia o aggiornamenti, occorre integrare il corpus e rivalutare.
```

## Use Rules

- Keep the answer concise.
- Do not add unsupported background knowledge.
- Do not invent source references.
- Prefer a narrower answer over a broad unsupported synthesis.
- Use Italian by default.
